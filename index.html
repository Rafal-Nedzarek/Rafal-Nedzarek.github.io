About Me

My Projects

My biggest struggle over the years has been to keep my programming a sustained effort. Having to juggle full-time work at the office with steady freelance translation contracts, I often had to postpone my learning to a later date.

I decided I would need to work on personal projects – not only to showcase my skills but to keep my focus. To finally connect various technologies I had learnt, I chose the Django framework. I started with the Python and Django Full Stack Web Developer Bootcamp course on Udemy and quickly supplemented it with official Django documentation, Stack Overflow and other sources.

I used MySQL RDBMS with the Employees Sample Database and based my ORM models on the existing tables with the instpectdb utility. I built a simple application to query the Employees table and display related data from other tables. My chief focus was the views.py file, so the front-end will be left bare-bones for the time being. Finally, I used pycodestyle to make my code more readable.

Right now I run Django locally, using the built-in development server. I plan to host my app on Heroku or AWS in the near future. Until then, you can find some of the files on my GitHub page. I will soon add a simple ModelForm to enable adding entries into some of the tables. I also plan to introduce some automated testing and to read more about the ORM to tweak my queries. Once I’m done with this project, I would like to refresh and broaden my JavaScript knowledge and delve into either Angular or React.
